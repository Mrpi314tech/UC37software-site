How to use

To install:

Press "The Download Now" button. It will download a file, Install.sh.
Navigate to this file and execute it. It will take 3-5 minutes to install.
You will have to answer some basic questions like your name.
It is now installed. It can be run from the menu, or with the command "UC37"

Note

If you type the @ sign and then text, it will run the text like a voice command.
Example: typing "@ hello" is the equivalent of saying "hello" to UC37
